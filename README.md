# UI-UX-Experiment6

## Aim:

To implement combined senario.

## Algorithm:

Select User Story – Choose the action the user wants to perform.

Identify Persona – Define who the user is.

Set Context – Describe where, when, and why the user starts the task.

Define Trigger – What initiates the user’s action.

List User Steps – Write the sequence of actions the user performs.

Add System Responses – What the system does after each step.

Add Decision Points – Any yes/no or alternative paths.

Describe Outcome – Final result (success/failure).

Summarize Combined Scenario – Merge all into one short narrative.
